## 🎯 核心亮点

### 1. 侧边栏设计（最大亮点）⭐

#### 展开状态（256px）
```
┌──────────────────────────┐
│  MetaHub          [<]    │  ← Header
├──────────────────────────┤
│   👤  username           │  ← User Info
│       email@example.com  │
├──────────────────────────┤
│   🏠  首页               │  ← Navigation
│   💬  会话               │
│   ⚙️  设置               │
├──────────────────────────┤
│                   🌙     │  ← Theme
│   🚪  退出登录           │  ← Logout
└──────────────────────────┘
```

#### 折叠状态（64px）
```
┌─────┐
│ [☰] │
├─────┤
│  👤 │
├─────┤
│  🏠 │
│  💬 │
│  ⚙️ │
├─────┤
│  🌙 │
│  🚪 │
└─────┘
```

#### 技术实现
- 使用 `useState` 管理折叠状态
- CSS `transition-all duration-300` 平滑动画
- 条件渲染文字内容
- 图标始终显示
- 按钮宽度和对齐方式动态调整

## 🔒 安全特性

### 密码安全
1. 前端使用 Web Crypto API 进行 SHA256 哈希
2. 密码不以明文传输
3. 后端再次使用 bcrypt 哈希存储

### Token 管理
- **Access Token**: 短期有效（30 分钟）
- **Refresh Token**: 长期有效（7 天）
- **自动刷新**: 401 错误时自动刷新
- **安全存储**: localStorage

### 路由保护
- `ProtectedRoute` 组件
- 未登录自动跳转
- 加载状态处理

## 🔄 API 集成

### 后端接口
- `POST /api/v1/auth/register` - 用户注册
- `POST /api/v1/auth/login` - 用户登录
- `POST /api/v1/auth/logout` - 用户登出
- `POST /api/v1/auth/refresh` - 刷新 Token
- `GET /api/v1/auth/me` - 获取用户信息

### 配置
```env
VITE_API_BASE_URL=http://localhost:8000
```
