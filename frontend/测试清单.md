## 1️⃣ 注册功能测试

### 正常流程
- [ ] 访问 `/register` 页面
- [ ] 填写用户名：`testuser`
- [ ] 填写密码：`Test@123456`
- [ ] 填写邮箱：`test@example.com`（可选）
- [ ] 填写手机号：`13800138000`（可选）
- [ ] 点击"注册"按钮
- [ ] 验证：注册成功后自动登录并跳转到首页

### 异常流程
- [ ] 用户名少于 3 个字符 → 显示错误提示
- [ ] 密码少于 8 位 → 显示错误提示
- [ ] 密码不包含大写字母 → 显示错误提示
- [ ] 密码不包含小写字母 → 显示错误提示
- [ ] 密码不包含数字 → 显示错误提示
- [ ] 密码不包含特殊字符 → 显示错误提示
- [ ] 两次密码不一致 → 显示错误提示
- [ ] 邮箱格式错误 → 显示错误提示
- [ ] 用户名已存在 → 显示后端错误提示
