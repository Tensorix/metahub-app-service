# MetaHub 前端项目说明

## 🎯 项目概述

这是一个基于 **React 19 + TypeScript + Vite + shadcn/ui** 构建的现代化前端应用，实现了完整的用户认证系统和优雅的侧边栏导航。

## ✨ 核心功能

### 1. 认证系统
- ✅ 用户注册（用户名、密码、邮箱、手机号）
- ✅ 用户登录（支持用户名/邮箱/手机号登录）
- ✅ 自动 Token 刷新（无感知续期）
- ✅ 安全退出登录
- ✅ 密码 SHA256 哈希处理

### 2. 优雅的侧边栏（重点功能）⭐
- ✅ **可折叠设计**：点击按钮在 256px 和 64px 之间切换
- ✅ **平滑动画**：300ms 过渡效果
- ✅ **用户信息**：显示头像、用户名、邮箱
- ✅ **导航菜单**：首页、会话、设置
- ✅ **当前页高亮**：自动识别当前路由
- ✅ **响应式设计**：适配各种屏幕尺寸
- ✅ **主题切换**：集成在侧边栏底部
- ✅ **退出登录**：快速访问

### 3. 深色模式支持 🌙
- ✅ **浅色模式**：明亮清爽的界面
- ✅ **深色模式**：护眼舒适的暗色主题
- ✅ **跟随系统**：自动适配系统主题设置
- ✅ **持久化**：设置保存到本地，刷新后保持
- ✅ **平滑切换**：主题切换带有过渡动画

### 4. 首页展示
- ✅ 大标题欢迎语："Hi, {username} 👋"
- ✅ 账户信息卡片（用户名、邮箱、手机号、状态）
- ✅ 快速开始引导
- ✅ 系统信息展示
- ✅ 响应式卡片布局

## 🎨 设计亮点

```
┌──────────────────────────┐
│  MetaHub          [<]    │  ← 应用名称 + 折叠按钮
├──────────────────────────┤

│       email@example.com  │     用户名 + 邮箱
├──────────────────────────┤
│   🏠  首页               │  ← 导航菜单
│   💬  会话               │     图标 + 文字

│                          │
├──────────────────────────┤
│                   🌙     │  ← 主题切换
```

### 主题系统

使用 CSS 变量实现主题切换：
- 定义浅色和深色两套颜色变量
- 通过切换 `.dark` 类名实现主题切换
- 所有组件自动响应主题变化
- 平滑的过渡动画

### 响应式设计

- **移动端**（< 768px）：单列布局，侧边栏可完全折叠
- **平板**（768px - 1024px）：2 列卡片布局
- **桌面**（> 1024px）：3 列卡片布局，侧边栏展开

## 🔐 安全特性

### 密码安全
1. 前端使用 Web Crypto API 进行 SHA256 哈希
2. 密码不以明文传输
3. 后端再次使用 bcrypt 哈希存储

### Token 管理
- **Access Token**：短期有效（30 分钟）
- **Refresh Token**：长期有效（7 天）
- **自动刷新**：Access Token 过期时自动刷新
- **安全存储**：使用 localStorage 存储

### 路由保护
- 使用 `ProtectedRoute` 组件保护需要登录的路由
- 未登录自动跳转到登录页
- 验证期间显示加载动画

## 📖 使用流程

### 注册新用户
1. 访问 `/register` 注册页面
2. 填写用户名（至少 3 个字符）
3. 填写密码（至少 8 位，需包含大小写字母、数字和特殊字符）
   - 示例：`Test@123456`
4. 可选填写邮箱和手机号
5. 点击"注册"按钮

### 登录系统
1. 注册成功后自动登录
2. 或访问 `/login` 手动登录
3. 支持使用用户名、邮箱或手机号登录

### 使用侧边栏
1. 点击左上角按钮折叠/展开侧边栏
2. 点击菜单项切换页面
3. 当前页面会高亮显示
4. 点击底部主题按钮切换深色/浅色模式
5. 点击"退出登录"按钮退出

### 查看首页
- 显示个性化欢迎语："Hi, {username} 👋"
- 查看账户信息卡片
- 浏览快速开始指南
- 查看系统信息

## 🎯 核心特性说明

### 1. 状态管理（Zustand）

**认证状态** (`store/auth.ts`)：
- 管理用户登录状态
- 存储用户信息
- 提供登录、注册、登出方法
- 自动初始化认证状态

**主题状态** (`store/theme.ts`)：
- 管理当前主题（light/dark/system）
- 提供主题切换方法
- 持久化到 localStorage

### 2. API 集成（Axios）

**功能**：
- 统一的 API 客户端
- 请求拦截器（自动添加 Token）
- 响应拦截器（自动刷新 Token）
- 完整的类型定义

**接口列表**：
- `POST /api/v1/auth/register` - 注册
- `POST /api/v1/auth/login` - 登录
- `POST /api/v1/auth/logout` - 登出
- `POST /api/v1/auth/refresh` - 刷新 Token
- `GET /api/v1/auth/me` - 获取用户信息

### 3. 路由系统（React Router）

**路由配置**：
```
/login              → 登录页（公开）
/register           → 注册页（公开）
/                   → 主布局（需登录）
  ├── /             → 首页
  ├── /sessions     → 会话页
  └── /settings     → 设置页
```
